One2many Mass Select Delete Widget
==================================
This module is used to create a widget that used for One2Many fields which is
used to delete selected order lines. similarly it also have the functionality to
delete the unselected lines.

Configuration
=============
- <field name="order_lines" widget="one2many_delete">

Installation
============
- www.odoo.com/documentation/17.0/setup/install.html
- Install our custom addon
